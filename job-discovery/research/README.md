# Job Discovery Research

This directory stores source research and planning notes for the job-discovery pipeline.

- `ats-query-chat.txt`: raw ChatGPT conversation export about using search operators, ATS-hosted career pages, and recency filters to find recently indexed jobs.

Research files are treated as source context, not executable code. Implementation rules and usage instructions belong in `job-discovery/README.md`, `AGENTS.md`, and the configuration files under `job-discovery/config/`.

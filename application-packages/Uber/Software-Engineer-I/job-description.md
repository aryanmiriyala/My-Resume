## Role Summary

The Software Engineer I role designs, develops, and tests software applications using Uber's technology stack. The position emphasizes backend development, scalable and resilient systems, API integration, production reliability, engineering productivity, and collaboration with Engineering, Product, and Operations leaders to launch products and improve marketplace efficiency.

## Responsibilities

- Design, develop, and test software applications.
- Build software components that integrate into fully functional software systems.
- Apply technical knowledge to evaluate complex data, developments, and projects.
- Recommend solutions to technical issues.
- Support business operations and improve engineering productivity.
- Solve production issues in product and system reliability.
- Collaborate with Engineering, Product, and Operations leaders.
- Launch new products and enhance marketplace efficiency.
- May telecommute.

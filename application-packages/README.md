# Application Packages

Each folder represents one specific company and role.

Expected structure:

```text
application-packages/<Company>/<Role>/
  job-description.md
  tailoring-notes.md
  resume.tex
  resume.pdf
  cover-letter.md
  cover-letter.pdf or cover-letter.docx
```

Application packages are created only after Aryan selects a role or provides a specific job description. Broad job search results belong in `job-search/`, not here.

## Who We Are

Notion is the collaborative AI workspace where teams and agents think together. It is building one place where knowledge, projects, meetings, and AI tools live side by side so work is faster, clearer, and less fragmented. Millions of individuals, small teams, and large companies run their work on Notion.

Notion employees are customer zero in bringing this future of work to life. Notion cares about craft, building things that last, and the belief that great work is still fundamentally human. Notion's goal is to set the standard for how humans work together in the AI era, from building a business's system of record to making and managing AI agents to automating away busy work.

## About The Role

As an engineer at Notion, new grads help shape core user experiences and accelerate how people discover value in Notion. Engineers tackle meaningful challenges with increasing autonomy, craft code that millions of users will experience, take ownership of meaningful projects, make critical technical decisions, and contribute to product vision. The role works with design, product, and data across several teams.

## What You'll Achieve

- Work with others to plan, shape, and build new product features from start to finish, through conception, research, implementation, and maintenance.
- Potentially work on adoption by instrumenting key onboarding moments and iterating on activation flows.
- Improve performance and reliability or polish existing features, such as syncing spell-check dictionaries across browsers or improving search to index file attachments.
- Build internal tools to support simplicity and productivity for the team, such as writing scripts to import user feedback into Notion.

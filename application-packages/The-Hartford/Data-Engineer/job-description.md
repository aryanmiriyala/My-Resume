## Company Context

The Hartford is an insurance company focused on helping customers, communities, and employees while building products and services that go beyond coverages and policies. For this role, the most relevant context is the company's actuarial, modeling, analytics, and insurance-data environment.

## Role Summary

The Hartford is seeking a Data Engineer within the Actuarial Strategic Modeling team to design, develop, and implement sustainable data assets that fuel machine learning and artificial intelligence solutions across strategic initiatives. The role emphasizes continuous data delivery, modern software engineering practices, MLOps, Agile delivery, well-architected data solutions, and cross-functional collaboration.

## Responsibilities

- Develop high-quality, scalable software modules for next-generation analytics solutions.
- Work with cross-functional IT stakeholders such as database, operations, and technical support teams.
- Translate business problems into logical statements and implement efficient, cost-effective application program solutions.
- Identify and validate internal and external data sources for availability, quality, lineage, and use-case suitability.
- Create data assets and data pipelines aligned with modern software development principles.
- Perform data analysis and exploratory checks for nulls, duplicates, and other data-quality issues.
- Assist with code that enables real-time modeling solutions to be ingested into front-end systems.
- Produce GitHub-based code artifacts and documentation for reproducible results and handoff to data science teams.

## Qualifications and Skills

- 2+ years of relevant experience recommended.
- Bachelor's degree in Computer Science, Engineering, IT, Management Information Systems, or related discipline.
- Experience in R preferred; Python and SQL required/relevant.
- Experience ingesting data from relational databases, Hadoop/Spark, cloud data sources, XML, and JSON.
- ETL experience involving metadata management and data validation.
- Linux and Git.
- Automation tools such as Autosys, Cron, or Airflow.
- Exposure to AWS services such as S3 and EMR is a plus.
- Ability to communicate with technical and non-technical teams.
- Ability to translate complex technical topics into business solutions and strategies.

# Nymbus - AI-First Software Engineer

Role: AI-First Software Engineer (Entry Level, multiple roles open)

Location: Remote - Atlanta, Georgia

Company: Nymbus

Website: https://nymbus.com/

## Company Summary

Nymbus is a fintech company focused on reimagining banking. Its modern core platform and cloud-based technology serve financial institutions that want to modernize. The company emphasizes collaboration, integrity, a client-first approach, reduced technical debt, agility for banks and credit unions, and an AI-first mindset across the business.

## Work Environment

Nymbus is primarily remote-first. Employees may need to travel a few times per year to client sites or designated team locations. Working hours align with the Eastern Time Zone. Eastern or Central time zone candidates are preferred, and priority is given to Atlanta-based candidates.

## Position Summary

Experience level: New College Graduate / Early Career (0-5 years)

Team: Product Engineering

## Why This Role Exists

Nymbus believes the next generation of great engineers will be AI-native, not AI-adjacent. The role is designed for a new or recent college graduate who treats AI as a core development tool. The engineer will work with senior engineers to ship real features, improve production systems, and experiment with AI-driven development workflows. Nymbus is looking for someone who takes ownership, moves fast, and cares about efficient design, code quality, and impact.

## What You Will Do

- Build and ship production features across backend, frontend, and AI-enabled workflows.
- Use AI tools such as Claude, AWS Kiro, Copilot, or similar tools as force multipliers, not crutches.
- Write clean, readable, testable code that can scale with the business.
- Collaborate with product management and design teams to turn ambiguous problems into working solutions.
- Prototype quickly, validate assumptions, and iterate based on real feedback.
- Improve developer productivity through automation, tooling, and experimentation.
- Learn systems deeply and continuously raise the quality bar.

## Core Traits

- High agency.
- Problem solver.
- Experimenter mindset.
- Quality-driven.
- Scale-oriented thinking.
- Personal ownership.

## Technical Foundation

- Strong fundamentals in computer science, including data structures, algorithms, and systems thinking.
- Experience building applications in at least one modern language such as Python, TypeScript, Java, or Go.
- Familiarity with APIs, databases, and basic cloud concepts.
- Comfortable using Git and collaborating in a team codebase.

## AI-First Bonus Points

- Hands-on experience with Claude, AWS Kiro, or similar AI developer tools.
- Curiosity about LLMs, prompt design, agent workflows, or AI-assisted architectures.
- Experience using AI for code generation and refactoring, test creation, debugging, and reasoning through complex logic.

## Success Profile

First 1-6 months:

- Learn the service delivery lifecycle and existing products.
- Become proficient in standard development tools and AI services for development.
- Complete multiple POCs and investigations in innovation projects.
- Troubleshoot issues using AI services.
- Grow faster than a traditional entry-level engineer.

First 6-12 months:

- Ship meaningful features independently.
- Proactively improve code, tooling, or workflows.
- Use AI intelligently, knowing when to trust it and when to override it.
- Conduct code reviews for other developers.
- Share lessons learned and tips for AI-assisted development.

## Interview Process

The interview process typically includes up to three interviews: HR discussion, hiring manager, and technical interview. The technical interview requires a demonstration of development using the candidate's preferred IDE and focuses on how the candidate has incorporated AI into development activities, including services used and how AI supported debugging, feature design, code review, and code generation.

## Salary and Benefits

Salary: $75,000 - $85,000 annually.

Benefits include annual cash bonus and equity options, 100% remote work, 401(k), health/dental/vision insurance, and flexible paid time off.

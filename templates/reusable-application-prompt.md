# Reusable New Application Prompt

Use this prompt when starting a new chat or handing Codex a new job description from this repository.

```text
Here is the next role I am applying to. Read AGENTS.md first and follow the full application pipeline exactly. Create the application folder, save the JD, tailor the resume and cover letter, generate PDFs, run the ATS/alignment pass, update the tracker, validate the package, commit, and push. Keep all claims grounded in my source materials.

Job description:
...
```

- Application link:
- Application questions:
- Company or product interest:
- Personal connection to the company, industry, mission, or team:
- Anything specific the cover letter should make the hiring manager feel:

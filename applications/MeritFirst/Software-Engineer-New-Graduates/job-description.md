## Company / Platform Context

MeritFirst is a curated talent network for high-growth startups, including 8VC and Slow Ventures portfolio companies. The application is intended to connect early-career engineers with companies actively hiring through a work-sample assessment rather than relying only on resume keyword screens.

The posting emphasizes building real software, taking a required 20-30 minute work-sample assessment, and being evaluated on ability to build, debug, explain work, and learn.

## Responsibilities

Depending on the team, engineers may:

- Build backend systems that power SaaS platforms.
- Work on frontend interfaces used by real customers.
- Solve infrastructure challenges that help companies scale.
- Debug production issues and ship fixes.
- Own features from design through deployment.
- Write real code, ship to production, and learn from experienced engineers.

## Requirements / Signals

- Know at least one programming language well enough to build working software.
- Understand fundamentals: data structures, how code runs, and debugging.
- Write code that works, not just code that is fast to write.
- Be curious about systems and tradeoffs.
- Take feedback without ego and improve every week.
- Explain work clearly and ask good questions.
- Strong project/building evidence is more important than school, GPA, or brand-name internships.

## Constraints / Risks

- The assessment is required and should be completed before or during submission.
- The posting is for a network of companies, so the resume should be broadly software-engineering focused rather than over-specialized for one stack.
- Hybrid locations include Austin, New York City, San Francisco, and Remote (United States); final location expectations may vary by matched company.
